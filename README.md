## Getting Started

### Installation

1. Clone the repo
     ```sh
     git clone https://github.com/rwndy/react-todo-app.git
     ```
  2. Install packages
     ```sh
     pnpm install
     ```
  3. Start Project
     ```sh
     pnpm run dev
     ```
